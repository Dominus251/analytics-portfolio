# Анализ игрового рынка Steam (2025)

## Цель
   Построить полный конвейер обработки данных: от загрузки сырых CSV до дашборда в Power BI.

## Источник данных
   Steam Dataset 2025 (Kaggle) — 13 таблиц, >1 млн отзывов. [Датасет](https://www.kaggle.com/datasets/crainbramp/steam-dataset-2025-multi-modal-gaming-analytics?resource=download-directory)

## Структура данных

   Проект использует следующие основные таблицы в PostgreSQL:

   - **applications** — основная таблица с информацией об играх (appid, name, цена, дата релиза, поддержка платформ).
   - **genres, developers, publishers, platforms, categories** — справочники.
   - **application_genres, application_developers, ...** — связующие таблицы (many-to-many).
   - **reviews** — пользовательские отзывы.

## Инструменты
  1. PostgreSQL (хранение и подготовка)
  2. DBeaver (работа с БД)
  3. Python + pandas (очистка reviews)
  4. ClickHouse (колоночная БД для дашбордов)

Power BI (финальная визуализация)

## План работы
   1. **Загрузка данных в PostgreSQL**  
      Импорт 13 CSV-файлов из Steam Dataset.  
      Проверка целостности и первичная очистка (удаление дублей, обработка NULL).

      Особенности загрузки reviews.csv

      Все таблицы были загружены через DBeaver с использованием встроенного импорта CSV, **кроме таблицы `reviews`** — из-за проблем с кавычками в CSV. Для неё использовался Python-скрипт [`load_reviews.py`](./python_scripts/load_reviews.py), который очищает данные и загружает их через SQLAlchemy.

      Скрипт выполняет:
      - чтение CSV с игнорированием некорректных строк,
      - фильтрацию только нужных колонок,
      - удаление строк с NULL в ключевых полях,
      - пакетную вставку в PostgreSQL.

   2. **Очистка и денормализация**  
      Объединение таблиц (applications, genres, developers, publishers, reviews).  
      Расчёт ключевых метрик: средний рейтинг, количество отзывов, суммарные продажи (если есть), распределение цен и жанров.  
      Создание широкой витрины для анализа.

   3. **Ad-hoc аналитика в Excel**  
      Точечные запросы для проверки гипотез.  
      Экспорт результатов в Excel для изучения.

   4. **Дашборды в Google Sheets**  
      1–2 интерактивных дашборда на основе агрегированных данных (например, динамика релизов по годам, распределение цен по жанрам).  
      Подключение к PostgreSQL через `=QUERY` или скрипты.

   5. **Экспорт витрины в ClickHouse**  
      Перенос денормализованной таблицы в колоночную БД для ускорения запросов.

   6. **Финальный дашборд в Power BI**  
      Подключение к ClickHouse.  
      Построение дашборда для директора с основными бизнес-метриками (обзор рынка, тренды, топ-игры, издатели).

   ### Возможности фильтрации

 На листе `Dashboard` вы можете фильтровать данные по следующим параметрам:
  - **Издатель** – выбор одного или нескольких издателей.
  - **Год релиза** – фильтр по конкретному году.
  - **Месяц релиза** – фильтр по месяцу.
  - **Тип продукта** – `game`, `dlc`, `demo` и т.д.
  - **Форма продаж** – `бесплатные` / `платные игры`.
  - **Поддержка платформ** – `Windows`, `Mac`, `Linux`.
  - **Возрастной рейтинг** – фильтр по `required_age`.

   ### Графики на дашборде

 - **Количество релизов по годам** – динамика выхода игр.
 - **Топ-50 издателей по количеству релизов** – кто выпускает больше всего игр.
 - **Топ-20 издателей по популярности** – измеряется по числу отзывов.

   ### Как использовать

   1. Выберите значения в выпадающих списках или введите текст в поля.
   2. Для множественного выбора доступны: **Издатель**, **Год релиза**, **Возрастной рейтинг** и **Тип продукта** .

## Денормализация таблицы и экспорта в ClickHouse с последующим экспортом в Power Bi

 Для ускорения запросов в ClickHouse и упрощения визуализации в Power BI была создана плоская витрина `steam_mart`, в которой:
- все связи `many-to-many` (издатели, жанры, разработчики, категории, платформы) свёрнуты в текстовые поля через `STRING_AGG`;
- агрегированы отзывы (`total_reviews`, `positive_share`);
- каждая игра представлена одной строкой (без дублирования).

 SQL-скрипт денормализации: [`steam_mart.sql`](./SQL/steam_mart.sql)

## Дашборд в Power BI

Финальный дашборд построен на основе витрины `steam_mart`, загруженной из **ClickHouse**.

 **Страницы дашборда содержат:**
 - Фильтры: год релиза, платформа, тип продукта, категория, возрастное ограничение.
 - График «Количество релизов по годам».
 - Доля бесплатных игр (20.14%).
 - Топ-5 издателей по количеству релизов.
 - Топ-5 жанров по количеству релизов.
 - Распределение по категориям.

**Интерактивность:**
 - Все графики связаны между собой.
 - Фильтры применяются ко всей странице.

 ### Бизнес-ценность дашборда

Дашборд помогает принимать стратегические решения в игровой индустрии:

- **Анализ рынка** — график «Количество релизов по годам» показывает тренды: когда рынок рос, когда насыщался. Это помогает планировать выход новых игр в периоды с наименьшей конкуренцией.
- **Оценка издателей** — топ-5 издателей по релизам и топ-10 по отзывам показывают, кто лидирует по количеству и качеству. Это позволяет формировать партнёрские стратегии и выбирать издателей для сотрудничества.
- **Жанровая аналитика** — топ-5 жанров по релизам и по взвешенному рейтингу помогают понять, какие жанры популярны, а какие — недооценены. Это основа для R&D: в каком жанре делать новую игру.
- **Монетизация** — доля бесплатных игр (20.14%) показывает, что Free-to-play — значимый сегмент. Это влияет на выбор бизнес-модели.
- **Качество продукта** — топ-5 игр по взвешенному рейтингу позволяет изучать успешные проекты: что делает их лучшими, какие механики работают.
- **Фильтры** — возможность смотреть данные в разрезе года, платформы, типа продукта, категории и возрастного ограничения позволяет быстро проверять гипотезы без написания SQL-запросов.

**Результат для бизнеса:** сокращение времени на аналитику, более обоснованные инвестиции в разработку и маркетинг, снижение рисков при запуске новых проектов.

## Выводы

- Рынок Steam демонстрирует устойчивый рост числа игр, особенно в сегменте инди-проектов.
- Бесплатные игры составляют ~20% от всех релизов.
- Топ-издатели (KOMODO, TigerQiuQiu, KOEI TECMO) удерживают лидерство по объёму релизов.
